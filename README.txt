# snake-game

Este es un proyecto básico del juego de la serpiente desarrollado en Python utilizando la librería pygame. 
Es parte de un trabajo universitario para aprender conceptos básicos de programación.

---

## Funcionalidades Actuales
- Pantalla de inicio con opciones:
  - Presiona ESPACIO para comenzar el juego.
  - Presiona ESC para salir del programa.
- Mensaje de transición antes de iniciar el juego.

---

## Requisitos del Sistema
1. Python:
   - Versión 3.6 o superior.
2. Librería pygame:
   - Instálala con el siguiente comando:

     pip install pygame
